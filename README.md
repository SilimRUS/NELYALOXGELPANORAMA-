# Panorama NELYA

Красивая галерея панорамных фотографий.

## О проекте

Этот проект представляет собой веб-галерею для демонстрации панорамных фотографий с адаптивным дизайном.

## Технологии

- HTML5
- CSS3
- Адаптивный дизайн
- GitHub Pages

## Как запустить

1. Склонируйте репозиторий
2. Откройте `index.html` в браузере
